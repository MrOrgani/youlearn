You learn is a Learning management system:
- Create a course
- Update your videos
- Share them with anyone
- Make money

# Context

It was a freelance project.

# Stack 

Made with: 
- NextJS
- TypeScript
- Tailwind
- Prisma
- Mux
- UploadThing
- Zod
- Axios
- Stripe
